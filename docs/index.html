Installing Chrome Extension from ZIP File

  1. Find and download the ZIP file of the extension from the "Assets" section.
  2. Open Google Chrome and go to the extensions management page:
    • Option 1: Type "chrome://extensions" in the address bar and press Enter.
    • Option 2: Click the menu icon (three vertical dots) at the top right, go to "More Tools", and select "Extensions".
  3. Enable the "Developer mode" toggle switch located in the top right corner of the extensions management page.
  4. Click the "Load unpacked" button that appears in the top left corner of the extensions management page.
  5. Select and open the downloaded ZIP file of the extension.
  6. The extension should now be installed and visible on the extensions management page.
